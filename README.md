# Projeto Paradigmas da Programação:

Trabalho para Paradigmas de Programação:


Proposta: 
Desenvolvimento de um web software para controle da produção de peças cortadas e dobradas em uma empresa metalúrgica.
O sistema permitirá o registro de ordens de produção, controle de materiais, cálculo de indicadores de desempenho, geração de relatórios automáticos e previsão de produção.


Objetivos:
Automatizar o processo de controle de produção e qualidade. Calcular indicadores como produtividade, taxa de defeitos e OEE. Permitir gestão de metas diárias. Gerar relatórios em tabela e PDF com gráficos. Implementar previsão de produção usando dados históricos. Fornecer diferentes níveis de acesso: administrador, gerente e funcionário.


Tecnologias Utilizadas:
Frontend: HTML, CSS, React (JavaScript), Material-UI, Axios, jsPDF + html2canvas, SignalR(atualização em tempo real).
Backend: C# (ASP.NET Core, Entity Framework Core, HttpClient, JWT, API REST (get/post), Conexão MySql).
Banco de Dados: MySQL,Bogus (geração de dados para teste).
Análise e Relatórios: Python (pandas, matplotlib, reportlab).
Módulo de IA: Flask, scikit-learn, Integração com C# via HTTP.
